Admin — Blog

Gerencie os artigos do seu blog de forma simples e segura

Abrir Editor de Artigos
Como funciona
  1. Abra o Editor de Artigos acima
  2. Preencha o título, slug, conteúdo (Markdown) e outras informações
  3. Clique em "Baixar Arquivo .md" — o arquivo será salvo no seu computador
  4. Copie o arquivo para a pasta content/blog/ do seu repositório
  5. Faça commit via Git: git add content/blog/ && git commit -m "Novo artigo" && git push
Por que esse método?

Este editor funciona 100% no seu navegador, sem precisar de servidor, banco de dados ou tokens de acesso. Os artigos ficam salvos no seu navegador (localStorage) e você baixa os arquivos .md para colocar no repositório Git.
