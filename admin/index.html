Admin — Blog

Editor local para artigos — sem token, sem servidor

Editor de Artigos

Abra a pasta content/blog/ do projeto, escreva com editor visual (estilo Google Docs) e salve os arquivos .md no disco. Depois faça commit manualmente com Git.

Abrir Editor
Como funciona
  1. Abra admin/editor.html no Chrome ou Edge (duplo clique no arquivo)
  2. Clique em Abrir pasta content/blog/ e selecione a pasta do projeto
  3. O navegador lembra a última pasta na próxima vez que abrir
  4. Crie ou edite artigos com o editor visual completo
  5. Clique em Salvar arquivo — grava {slug}.md e atualiza articles.json
  6. Faça commit e push quando quiser: git add content/blog/ && git commit -m "..." && git push
Por que local?

Nada de token GitHub no navegador. Os arquivos ficam na sua máquina, no mesmo lugar do repositório. Você controla quando publicar no site.
