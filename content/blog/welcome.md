# Olá, visitante!

Este é o primeiro artigo publicado no blog. Aqui compartilho reflexões sobre **tecnologia**, **sistemas** e o dia a dia de quem desenvolve software.

## Por que um blog estático?

O conteúdo vive no repositório Git em Markdown com metadados YAML. Isso permite versionar textos, publicar via GitHub Pages e manter o site sem banco de dados ou servidor de aplicação.
